# מעבדת בדיקות - תמלול וקלדן (פרויקט עצמאי)

פרויקט Flask **נפרד לחלוטין** מהמערכת הראשית. אין כאן שום קשר לימות המשיח,
ללקוחות, לחיוב, או לבסיס הנתונים של המערכת הראשית. אפשר לפרוס ולעדכן את
הפרויקט הזה מתי שרוצים, בלי שום סיכון לשבש שירות ללקוחות קיימים.

## מה יש כאן
- `app.py` — שרת Flask עם דף העלאה יחיד (`/`) וטיפול בבקשה (`/run`)
- `engines.py` — כל הלוגיקה של תמלול (Gemini / AlefBot) ו-OCR לכתב יד (Gemini / Claude / GPT-4o)
- `requirements.txt` — כל התלויות

## התקנה ופריסה ב-Railway
1. תיצור repo חדש ב-GitHub, תעלה את כל 3 הקבצים (`app.py`, `engines.py`, `requirements.txt`)
2. ב-Railway: New Project → Deploy from GitHub repo → תבחר את הריפו החדש
3. ב-Variables של הפרויקט תוסיף:

| משתנה | ערך |
|---|---|
| `LAB_ACCESS_CODE` | קוד סודי שאתה בוחר - זה מה שתקליד בטופס בשדה "קוד גישה" |
| `LAB_DEFAULT_EMAIL` | המייל שאליו יישלחו תוצאות כברירת מחדל |
| `APP_BASE_URL` | הכתובת הציבורית של הפרויקט הזה עצמו ברגע שיהיה לו דומיין, למשל `https://transcribe-lab-production.up.railway.app` (בלי `/` בסוף) |
| `GOOGLE_API_KEY` | מפתח Gemini לתמלול אודיו/וידאו |
| `GOOGLE_API_KEY_OCR` | מפתח Gemini ל-OCR כתב יד (אפשר להשתמש באותו מפתח כמו למעלה אם אין לך נפרד) |
| `ANTHROPIC_API_KEY` | מפתח Claude, לבדיקת OCR עם Claude |
| `OPENAI_API_KEY` | מפתח OpenAI, לבדיקת OCR עם GPT-4o |
| `ALEFBOT_API_KEY` | מפתח AlefBot, לבדיקת תמלול פרימיום |
| `SENDGRID_API_KEY` | מפתח SendGrid לשליחת מייל התוצאה |
| `SENDGRID_FROM_EMAIL` | כתובת "מאת" בשליחת המייל |

4. Railway יזהה אוטומטית שזה פרויקט Python (בזכות `requirements.txt`) ויריץ אותו.
   אם צריך `Start Command` ידני, זה: `gunicorn app:app`

## שימוש
אחרי הפריסה, תיכנס ל-`https://<דומיין הפרויקט>/`, תקליד את קוד הגישה,
תעלה קובץ, תבחר מנוע (או תשאיר "אוטומטי"), ותקבל את התוצאה למייל תוך כמה דקות.

## הערה על דיוק ה-OCR/תמלול
המנועים כאן הם גרסה **מפושטת** של המנועים במערכת הראשית - אותם מודלים
ואותו פרומפט בסיסי, אבל בלי הלוגיקה המורכבת של פיצול קבצים ארוכים לחלקים
ועיבוד מקבילי שקיימת שם. לרוב מקרי הבדיקה זה מספיק; לקבצים ארוכים במיוחד
(מעל כ-15 דקות אודיו) מומלץ להשתמש במנוע AlefBot, שמטפל בקבצים ארוכים מטבעו.
